# দক্ষিণ গোবধা পাবলিক লাইব্রেরী - APK তৈরির নির্দেশিকা

এই প্রজেক্টটি একটি আধুনিক রিঅ্যাক্ট (React) অ্যাপ যা ফায়ারবেস (Firebase) দিয়ে তৈরি। এই অ্যাপটি থেকে নিজের মোবাইলের জন্য APK ফাইল তৈরি করতে নিচের ধাপগুলো অনুসরণ করুন:

### ধাপ ১: প্রজেক্টটি ডাউনলোড করুন
- এই উইন্ডোর উপরে **Settings** (গিয়ার আইকন) এ ক্লিক করুন।
- **Export to ZIP** বাটনে ক্লিক করে পুরো প্রজেক্টটি আপনার কম্পিউটারে ডাউনলোড করুন এবং আনজিপ করুন।

### ধাপ ২: প্রয়োজনীয় সফটওয়্যার ইনস্টল করুন
আপনার কম্পিউটারে নিচের সফটওয়্যারগুলো থাকতে হবে:
১. **Node.js**: [https://nodejs.org/](https://nodejs.org/) থেকে ডাউনলোড করুন।
২. **Android Studio**: APK বিল্ড করার জন্য এটি প্রয়োজন (SDK সহ)।

### ধাপ ৩: কোড বিল্ড করুন
আপনার কম্পিউটারের টার্মিনাল বা কমান্ড প্রম্পটে (CMD) প্রজেক্ট ফোলডারে গিয়ে নিচের কমান্ডগুলো লিখুন:
```bash
npm install
npm run build
```

### ধাপ ৪: অ্যান্ড্রয়েড প্রজেক্ট তৈরি করুন
নিচের কমান্ডগুলো দিয়ে অ্যান্ড্রয়েড ফাইলগুলো সেটআপ করুন:
```bash
npx cap add android
npx cap sync
```

### ধাপ ৫: APK তৈরি করুন
- `npx cap open android` কমান্ডটি দিন, এতে **Android Studio** ওপেন হবে।
- অ্যান্ড্রয়েড স্টুডিওর ভিতরে **Build > Build Bundle(s) / APK(s) > Build APK(s)** এ ক্লিক করুন।
- কিছুক্ষণের মধ্যেই আপনি আপনার কাঙ্ক্ষিত APK ফাইলটি পেয়ে যাবেন।

---
**বি.দ্র.:** আপনি এই অ্যাপটি সরাসরি ব্রাউজারেও ব্যবহার করতে পারেন। যদি আপনি প্লে-স্টোরে পাবলিশ করতে চান, তবে আপনাকে একটি গুগল প্লে ডেভেলপার অ্যাকাউন্ট তৈরি করতে হবে।
