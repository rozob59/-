# মোবাইল এবং এআইডিই (AIDE) এর জন্য কোডিং নির্দেশিকা

আপনি চাইলে এই লাইব্রেরী অ্যাপটি একটি মাত্র ফাইল হিসেবে ব্যবহার করতে পারেন।

### ১. মাস্টার কপি (MASTER_COPY.html)
এই ফাইলটি একটি "All-in-one" ফাইল। এর ভেতরেই HTML, CSS, এবং JavaScript রয়েছে।
- **কিভাবে চালাবেন:** এই ফাইলটি সেভ করে যেকোনো ব্রাউজারে ওপেন করলেই অ্যাপটি কাজ করবে।
- **ফিচারসমূহ:**
  - ফায়ারবেস লগইন (Firebase Login)
  - বইয়ের তালিকা এবং সার্চ সিস্টেম
  - ইউজার এবং এডমিন প্যানেল আলাদা
  - **অটো নোটিফিকেশন:** বই ফেরত দেওয়ার তারিখের আগের দিন অ্যাপে অটোমেটিক নোটিফিকেশন ব্যানার আসবে।

### ২. এআইডিই (AIDE) তে নেটিভ কোড
আপনি যদি পিউর অ্যান্ড্রয়েড স্টুডিও বা এআইডিই তে নেটিভ জাভা (Java) দিয়ে বানাতে চান, তবে `AIDE_NATIVE_GUIDE.md` ফাইলটি দেখুন। সেখানে জাভা কোড এবং এক্সএমএল লেআউট দেওয়া আছে।

### ৩. ফাইল ডাউনলোড
আপনি যদি এই পুরো প্রজেক্টটি ডাউনলোড করতে চান:
১. উপরে **Settings** এ ক্লিক করুন।
২. **Export to ZIP** বাটনে ক্লিক করুন।

এর ফলে আপনার কাছে সব কোড একত্রে থাকবে।
